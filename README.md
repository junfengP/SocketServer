# SocketServer

#### 介绍
用于解决JAVA C/C++ C# Python不同语言之间数据交换的方法



#### 安装教程

1. 安装JRE 1.8
2. 下载zip包并解压
3. cd至解压目录
4. java -jar SocketServer.jar

#### 使用说明

1. 默认地址端口 127.0.0.1:5656
2. 内含两条命令，见[说明](./docs/Usage.md)
3. 指定监听端口(如5657): `java -jar SocketServer.jar 5657`
